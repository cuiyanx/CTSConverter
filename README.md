# nnTestTransfer
Transfer [nn](https://android.googlesource.com/platform/frameworks/ml) test case to [webml](https://github.com/intel/webml-polyfill) test case

## Prerequisites
* Need `python3`

## Start

```sh
$ npm start
```

## Passed JS test case
* `add.js`
* `mul.js`
* `concat_float_1.js`
* `concat_float_2.js`
